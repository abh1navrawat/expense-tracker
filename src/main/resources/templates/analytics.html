Asynchronous POI Generation Pipeline

Generating heavy financial records with Pivot Tables can strain thread lifecycles. Clicking the compilation button above dispatches a background task running in an isolated thread pool using Spring's @Async worker. It returns an HTTP 202 Accepted immediately, allowing you to browse the app while the compilation processes safely in the background. The UI polls the progress and downloads the spreadsheet once completed.
